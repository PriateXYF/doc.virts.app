数列极限

02.数列极限 - 思维导图

￼

用定义求数列极限的方法。

三部曲：

  1. 先写距离 \(|X_n - a| \lt \epsilon\)
  2. 反解出 \(n\) 的距离 : \(n \gt g(\epsilon)\)
  3. 取 \(N = [g(\epsilon )] + 1\) \((n \gt N)\)

收敛数列的三大性质

  1. 唯一性 : 给出数列 \(\{x_n\}\) ，若 \(\displaystyle\lim_{n\to \infty}x_n=a\) (存在)，则 \(a\) 是唯一的。
  2. 有界性 : 若数列 \(\{x_n\}\) 极限存在，则数列 \(\{x_n\}\) 有界。
  3. 保号性 : 设数列 \(\{x_n\}\) 存在极限 \(a\) ，且 \(a\gt0\) ( 或 \(a\lt0\) )，则存在正整数 \(N\) ，当 \(n\gt N\) 时，有 \(x_n\gt0\) ( 或 \(x_n\lt0\) ) 。

保号性的推论 : 如果数列 \(\{x_n\}\) 从某项起有 \(x_n\ge 0\) ，且 \(\displaystyle\lim_{n\to\infty}x_n=a\) ，则 \(a \ge 0\) 。

需要注意的是上述推论可以推出等号，例如 : \(\cfrac{1}{n} \gt 0\) 恒成立，但 \(\displaystyle\lim_{n\to\infty}\cfrac{1}{n} = 0\) 。

夹逼准则的定义

如果数列 \(\{x_n\}\) , \(\{y_n\}\) 及 \(\{z_n\}\) 满足下列条件

  1. \(y_n \le x_n \le z_n\) \((n = 1,2,3,\dots )\)
  2. \(\displaystyle\lim_{n \to \infty}y_n = a\) , \(\displaystyle\lim_{n \to \infty}z_n = a\)

则数列 \(\{x_n\}\) 的极限存在，且 \(\displaystyle\lim_{n \to \infty}x_n = a\)

夹逼准则不用验证等号。

单调有界准则的定义

单调有界数列必有极限，即若数列 \(\{x_n\}\) 单调增加(减少)且有上界(下界)，则 \(\displaystyle\lim_{n \to \infty}x_n\) 存在。

见到递推式 \(a_{n+1} = f(a_n)\) ，一般用单调有界准则。

判断数列单调递增/递减的方法。

  1. \(X_{n+1} - X_{n} > || < 0\)
  2. \(\cfrac{X_{n+1}}{X_{n}} > || < 1\) ( 同号 )

判断数列是否有界的方法。

  1. 用常规的不等式进行证明。
  2. 用放缩法进行判断。
  3. 用裂项相消(+放缩)法进行判断。
